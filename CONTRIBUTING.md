# 🤝 Contributing to repomap

Every PR helps unlock **Pull Shark** 🦈 — thank you!

## Quick Steps
```bash
gh repo fork yourusername/repomap --clone
cd repomap
git checkout -b feat/your-change
git commit -m "feat: describe change"
git push origin feat/your-change
gh pr create --fill
```
